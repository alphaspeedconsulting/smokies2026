# Event Images

This directory contains images for the Smokies 2026 event website.

## Image Descriptions

1. **banner-mountain-road.jpg** - Main banner: Corvettes driving on winding mountain road through autumn foliage
2. **attendees-car-meet.jpg** - Attendees section: Sports cars (Lamborghini, Corvettes) at car show/gathering
3. **logistics-parked-cars.jpg** - Logistics section: Line of Porsches parked at car meet
4. **routes-scenic-drive.jpg** - Routes section: Sports cars on winding road with autumn foliage
5. **schedule-car-gathering.jpg** - Schedule section: Porsches and sports cars at outdoor gathering

## Image Sources

Images are currently using Unsplash URLs. To use local images:
1. Download images matching the descriptions above
2. Place them in this directory
3. Update the HTML file to use local paths: `images/filename.jpg`

## Current Implementation

The HTML file uses Unsplash URLs that match the image descriptions. These can be replaced with local images by updating the `src` attributes in `index-integrated.html`.
